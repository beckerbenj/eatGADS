# Apply recodes according to a numerical check data.frame.

Applies recodes as specified by a `numCheck` `data.frame`, as created by
[`createNumCheck`](https://beckerbenj.github.io/eatGADS/reference/createNumCheck.md).

## Usage

``` r
applyNumCheck(GADSdat, numCheck)
```

## Arguments

- GADSdat:

  A `GADSdat` object.

- numCheck:

  A `data.frame` as created by
  [`createNumCheck`](https://beckerbenj.github.io/eatGADS/reference/createNumCheck.md).

## Value

A recoded `GADSdat`.
